# Tutorial-OWASP-ZAP
Guía completa para instalar y configurar OWASP ZAP

**Enlace de descarga** https://www.zaproxy.org/download/ y elegiremos según S.O
**Aquí trataremos la instalación en Kali Linux y el archivo Linux Installer**

-> Una vez descargado el archivo, abriremos **Terminal**
```bash
sudo su
```
-> Iremos al donde esté el archivo descargado
```bash
cd Downloads 
```
-> Le daremos permiso de ejecutable al archivo
```bash
chmod +x ZAP_2_14_0_unix.sh 
```
-> Ejecutamos archivo
```bash
./ZAP_2_1$_0_unix.sh
```
**Saldrá una guía de instalación, seguimos los pasos por defecto, y listo**
